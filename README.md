# Music-Recommendation-Journal
This repo contains a blueprint of an application with ASP.NET and C# to build a Music Recommendation Journal.

A personal mood tracking and music recommendation web app built with ASP.NET Core MVC.  
Log your moods, track intensity, add notes, and get suggested music genres based on your feelings. View statistics, trends, and export your mood data as JSON.  

## Features
- Log moods with intensity, notes, and tags
- Automatic genre suggestions per mood
- View mood statistics and intensity trends
- Filter and search mood entries
- Export data in JSON format

## Tech Stack
- ASP.NET Core MVC
- C#  
- Razor Views  
- In-memory storage (can be extended to database)
